supplement (Appendix 3): the English LaTeX source of all analytical proofs,
the Lean 4 machine-checked companion, and build scripts. All documentation is
in English.

2. REQUIREMENTS
--------------------------------------------------------------------------------

Lean verification (required for formal replication):
  - Lean 4 via elan (https://github.com/leanprover/elan)
  - lake (bundled with Lean 4)
  - Network access on first build (Mathlib is downloaded automatically)
  - Estimated first-build time: 15--45 minutes depending on machine and cache

Document compilation (optional):
  - TeX Live 2024 or later with pdflatex and bibtex

--------------------------------------------------------------------------------
3. HOW TO VERIFY THE LEAN FORMALIZATION
--------------------------------------------------------------------------------

Step 1 -- One-click build (recommended):
  From a terminal, run

      cd "/path/to/Appendix_3"
      chmod +x replicate.sh
      ./replicate.sh

  A successful run ends with "VERIFICATION COMPLETE" and leaves the full log in
  lean/build.log. The build must finish with no errors and no sorry obligations
  in Appendix3Proofs.lean.

Step 2 -- Manual build:
  cd lean
  lake build

Step 3 -- Read the analytical proofs (optional):
  Open tex/Appendix_3.tex or compile Appendix_3.pdf from tex/:

      cd tex
      pdflatex Appendix_3
      bibtex   Appendix_3
      pdflatex Appendix_3
      pdflatex Appendix_3

--------------------------------------------------------------------------------
4. WHAT THE LEAN FILE VERIFIES
--------------------------------------------------------------------------------

Appendix3Proofs.lean mirrors Sections S1--S8 of Appendix_3.tex:

  S1  Results from the main paper (I(0) closure, trend equivalence)
  S2  Public information and Bayesian learning
  S3  Stochastic implementation mechanism (MDG, Cox algebra, predictive masses)
  S4  Asymptotic type identification
  S5  Competitive risks and operational pressure
  S6  Policy implications (IFT step as explicit hypothesis)
  S7  Equilibrium implementability (fixed point as hypothesis)
  S8  Abstract Kakutani identification criterion

Declarations tagged [Proved] are machine-checked. Declarations tagged
[Hypothesis] are classical inputs proved analytically in the TeX supplement
(Kakutani mutual singularity, Blackwell--Dubins, Jaynes entropy, etc.).
See Section "Formal verification in Lean 4" in Appendix_3.tex and Table
"Lean formalization status for selected supplement objects."
